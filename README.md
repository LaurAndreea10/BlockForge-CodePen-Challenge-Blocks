# BlockForge 3.0 Ultimate

## English

BlockForge is a mobile-first 3D building and demolition game created for the CodePen Blocks Challenge.

### Game modes

Build, Demolish, Challenge, Campaign, Survival, Versus, Blueprint, Puzzle and Boss.

### Highlights

- Seven special block materials and six demolition tools
- Structural stability, gravity, damage, repair and chain reactions
- Campaign objectives, daily challenge and procedural levels
- Local multiplayer, profile, XP, achievements and leaderboard
- Blueprint accuracy, AI Architect analysis and replay
- Six-slot gallery, JSON import/export, PNG export and share links
- English/Romanian interface
- High contrast, large text, reduced motion, keyboard and gamepad support
- Installable PWA with offline support
- Responsive mobile controls and touch-friendly panels

### Controls

- Tap/click a grid cell to place a block
- Drag empty stage space to orbit
- Mouse wheel to zoom
- `G` toggles gravity
- `Delete` removes selected blocks
- `Ctrl + Z` undo
- Gamepad: A places, B opens demolition, X toggles gravity

### Tests

Open `tests.html` to run the browser smoke tests.

## Română

BlockForge este un joc 3D de construcție și demolare, optimizat pentru mobil și creat pentru provocarea CodePen Blocks.

### Moduri de joc

Construire, Demolare, Provocare, Campanie, Supraviețuire, Duel, Schiță, Puzzle și Boss.

### Funcții principale

- Șapte materiale speciale și șase unelte de demolare
- Stabilitate structurală, gravitație, deteriorare, reparare și reacții în lanț
- Obiective de campanie, provocare zilnică și niveluri procedurale
- Multiplayer local, profil, XP, realizări și clasament
- Precizie pentru schițe, analiză AI Architect și reluări
- Galerie cu șase sloturi, import/export JSON, export PNG și linkuri de distribuire
- Interfață română–engleză
- Contrast ridicat, text mărit, reducerea animațiilor, tastatură și gamepad
- PWA instalabilă, cu funcționare offline
- Controale tactile și interfață responsive
